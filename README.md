# 🏋️ Forge Gym Website

## 🌐 Live Demo  
👉 https://forgegym07.netlify.app/

---

## 📌 Overview
A modern and responsive gym website designed to showcase fitness services, trainers, and pricing plans.  
Built as a portfolio project to demonstrate web development and deployment skills.

---

## 🚀 Features
- Clean and modern UI  
- Hero section with call-to-action  
- About section  
- Services offered  
- Trainers showcase  
- Pricing plans  
- Contact section  
- Fully deployed online  

---

## 🛠 Tech Stack
- HTML  
- CSS  
- Netlify (Deployment)

---

## ⚡ Deployment
This project is deployed using Netlify with automatic updates via GitHub integration.

---

## ▶️ Run Locally
1. Download or clone the repository  
2. Open `index.html` in your browser  

---

## 💡 Description
This project represents a basic business website for a gym, demonstrating the ability to design and deploy real-world web pages for clients such as gyms, cafes, and local businesses.
